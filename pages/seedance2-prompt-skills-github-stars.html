ひとことで言うと

GitHub上で「Seedance 2.0 / prompt / skill」系の候補リポジトリを調べた範囲では、 2026年6月24日時点で最もスター数が多かったのは dexhunter/seedance2-skill でした。 GitHub APIで確認したスター数は 2,217 stars です。 2位は songguoxs/seedance-prompt-skill の 2,006 stars でした。

調査対象の前提

今回は、質問文の「promp skills」を「prompt skills / プロンプト作成用スキル」と解釈しました。 GitHubで「Seedance 2.0」「prompt」「skill」に関連する公開リポジトリを検索し、 そのうち説明文やREADME上でSeedance 2.0のプロンプト作成・Claude skill・Higgsfield向けPrompt Skillなどを扱うものを候補にしました。

  • 対象：Seedance 2.0の動画生成プロンプト作成を支援するGitHubリポジトリ。
  • 指標：GitHub APIの `stargazers_count`。
  • 時点：2026年6月24日時点。
  • 注意：GitHubのスター数は変動するため、将来順位が変わる可能性があります。

トップの dexhunter/seedance2-skill は何？

`dexhunter/seedance2-skill` は、Seedance 2.0で良い動画生成プロンプトを書くためのエージェント向けスキルです。 README上では、Claude Code、Cursor、Clineなど互換エージェントで使えるPrompt Writing Skillsとして説明されています。 英語版と中国語版のスキルファイルがあり、入力制約、参照構文、カメラ表現、プロンプト構造、テンプレートなどを扱う方向のリポジトリです。

なぜ「最もスターが多い」と言える？

今回の候補リストでは、1位の `dexhunter/seedance2-skill` が 2,217 stars、 2位の `songguoxs/seedance-prompt-skill` が 2,006 stars でした。 差は 211 stars なので、少なくとも今回確認したSeedance 2.0 prompt skill系候補の中では dexhunter/seedance2-skill がトップと判断できます。

ただし、GitHub検索は言葉の揺れに影響されます。 「Seedance」「Higgsfield」「Claude skills」「video prompt」など別キーワードで隠れた候補が見つかる可能性はあるため、 完全な全GitHub網羅というより、検索可能な主要候補の比較として読むのが安全です。

Prompt skillとして見るポイント

見る点理由
テンプレートの具体性カメラ、被写体、照明、動き、尺などをどこまで構造化しているかが重要。
対象ツールClaude Code、Cursor、Cline、Higgsfieldなど、どの環境向けかで使い勝手が変わる。
参照素材の扱い画像・動画・音声参照をどうプロンプトへ組み込むかがSeedance 2.0では重要。
更新頻度動画生成モデルは仕様が変わりやすく、古いプロンプト作法が合わなくなることがある。
スター数人気や発見されやすさの目安。ただし品質そのものの保証ではない。

スター数だけで選んでいい？

スター数は便利な人気指標ですが、必ずしも自分に一番合うとは限りません。 たとえばHiggsfield向けに特化したいなら、`beshuaxian/higgsfield-seedance2-jineng` や `OSideMedia/higgsfield-ai-prompt-skill` のようなHiggsfield寄りのリポジトリも候補になります。 一方、Seedance 2.0の基本的なプロンプト作成スキルを探すなら、今回トップの `dexhunter/seedance2-skill` はまず見る価値があります。

  1. Tavilyで `site:github.com seedance 2.0 prompt skills stars` を検索。

つまり何が大事？

現時点で「GitHubで最もスターを稼いでいるSeedance 2.0のprompt skillは？」と聞かれたら、 答えは dexhunter/seedance2-skill です。 ただし、これは2026年6月24日時点のスター数に基づく判断であり、スター数は日々変動します。 実際に使うなら、スター数に加えて、READMEの内容、更新日、対応エージェント、テンプレートの質も確認するとよいです。
